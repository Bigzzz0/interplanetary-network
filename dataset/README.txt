# Dataset Folder
# ==============
# 
# Place your video files in this folder to use them as the video source.
#
# Supported formats:
# - MP4 (.mp4)
# - AVI (.avi)
# - MOV (.mov)
# - MKV (.mkv)
# - WebM (.webm)
# - FLV (.flv)
# - WMV (.wmv)
#
# The sender will automatically use the first video file it finds.
#
# Example video sources:
# 1. Download sample videos from:
#    - https://sample-videos.com/
#    - https://www.pexels.com/videos/
#    - https://pixabay.com/videos/
#
# 2. Or record your own using the webcam and save here
#
# Recommended video specs:
# - Resolution: 640x480 or higher (will be resized)
# - Frame rate: 15-30 FPS
# - Duration: 10-60 seconds (will loop automatically)
